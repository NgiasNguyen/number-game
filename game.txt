MO TA GAME: HOC VIET SO CUNG AI

1) MUC DICH DU AN
- Xay dung mini-game giao duc cho tre em (4-7 tuoi) de luyen viet chu so tu 0 den 9.
- Ung dung AI nhan dien chu so viet tay de tao phan hoi tuc thi va tang hung thu hoc tap.
- Ho tro giao vien/phu huynh theo doi tien bo cua be thong qua cac chi so thong ke va bao cao theo tung so.

2) CONG NGHE SU DUNG
- Frontend: HTML, CSS, JavaScript (khong dung backend, khong framework).
- AI: TensorFlow.js, tai model local tai duong dan ./model/model.json.
- Model weights: ./model/group1-shard1of1.bin.
- Chay truc tiep tren trinh duyet qua local server, co the deploy tren Vercel va embed vao Google Sites.

3) CACH GAME HOAT DONG
- He thong random so muc tieu (0-9): "Hay viet so X".
- Be ve so tren canvas bang chuot (hoac but cam ung tren thiet bi ho tro).
- Co net dut goi y de be biet hinh dang so can viet.
- Nut "Xoa" de ve lai.
- Nut "AI Cham Diem" de gui bai ve cho AI danh gia.

4) CACH SU DUNG AI
4.1 Tai model
- Khi mo game, he thong tai model TensorFlow.js tu file local.
- Trang thai hien thi:
  + "Dang tai AI..."
  + "AI da san sang!"

4.2 Tien xu ly anh (tuong thich MNIST)
- Lay anh tu canvas cua be.
- Resize ve 28x28.
- Chuyen xam (grayscale).
- Chuyen sang gia tri 0-1 bang chuan hoa.
- Tao tensor dau vao co shape [1, 28, 28, 1].

4.3 Du doan
- AI tra ve xac suat cho 10 lop so (0-9).
- So du doan la lop co xac suat lon nhat.
- Do tin cay la gia tri xac suat lon nhat (phan tram %).

5) CACH CHAM DIEM
- Neu AI doan dung so muc tieu:
  + Diem lan nay = Do tin cay / 10
  + Vi du 87% => 8.7 diem
- Neu AI doan sai so muc tieu:
  + Diem lan nay = 0
- Ket qua hien thi bang popup giua man hinh:
  + Thong diep: "Tuyet voi!" hoac "Sap duoc roi, thu lai nhe!"
  + AI doan so cua ban
  + So dung
  + Diem cua ban

6) LOGIC THONG KE TONG QUAN
- Tong diem: tong diem tat ca cac lan cham.
- So luong so dung: tong so lan ve dung muc tieu.
- So luong so sai: tong so lan ve sai muc tieu.
- Do chinh xac trung binh (tong quan):
  + Chi tinh tren cac lan dung
  + Cong thuc: tong do tin cay cac lan dung / so lan dung
- Cau khich le:
  + Duoc sinh dua tren ty le dung va do chinh xac trung binh.

7) LOGIC REPORT THEO TUNG SO (0-9)
Moi so co cac chi so:
- Lan viet dung
- Lan viet sai
- Tong so lan cham cua so do
- Tong do chinh xac de tinh trung binh theo quy tac:
  + Lan dung: cong do tin cay AI
  + Lan sai: cong 0

Cong thuc do chinh xac TB tung so:
- Do chinh xac TB = tong do chinh xac / tong so lan cham cua so do

Vi du:
- So 3 duoc cham 4 lan:
  + Dung 2 lan voi 90% va 70%
  + Sai 2 lan => 0 va 0
  + Do chinh xac TB = (90 + 70 + 0 + 0) / 4 = 40%

8) QUY TAC TO MAU BANG BAO CAO
- Cot "Lan viet dung":
  + >= 1 => mau xanh
- Cot "Lan viet sai":
  + >= 1 => mau do
- Cot "Do chinh xac TB":
  + > 50 => mau xanh
  + < 50 => mau do
  + = 50 => mau mac dinh
  + Neu 0.0% nhung chua co du lieu (chua cham lan nao) => giu mau mac dinh

9) GIA TRI GIAO DUC
- Tre duoc luyen viet so bang hoat dong truc quan (ve tay).
- Nhan phan hoi nhanh, de hieu, tao dong luc hoc.
- Co theo doi tien bo theo tung so de tap trung luyen cac so yeu.

10) KET LUAN
- Game dat muc tieu ket hop hoc viet so va AI trong mot san pham nhe, de trien khai.
- Kien truc front-end thuan, de mo rong them am thanh, animation, badge thuong, hoac luu lich su hoc tap trong cac phien ban tiep theo.
